GIT

Como instalar GIT en Windows, Linux y Mac.

Git es un software de control de versiones diseñado por Linus Torvalds, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.

Windows

Descargar desde la página oficial de GIT la versión que nos corresponda, ya sea 32-bit o 64-bit.

Después de instalar, revisar versión con: 

1

git --version

Salida esperada: 

1

git version 2.33.1.windows.1

Linux

En Linux va a depender la distro que se está utilizando, aquí está la web de GIT con todos los comandos para instalación según la distro .

Igual dejo la versión de Ubuntu como ejemplo. Actualizar lista de paquetes con apt-get update y con sudo si es necesario. 

1

sudo apt-get install git

Revisar versión con: 

1

git --version

Salida esperada: 

1

git version 2.25.1

Mac

En los sistemas Mac se puede utilizar Homebrew, ver como instalar Homebrew aquí .

Instalar GIT utilizando Homebrew con el siguiente comando: 

1

brew install git

Revisar versión con: 

1

git --version

Salida esperada: 

1

git version 2.25.1

🔗Web Oficial de GIT para documentación más detallada.
